Colocando Imagens no Site:

Para colocar uma imagem basta digitar "img" que
o código será feito altomaticamente!

Para incluir uma imagem que está dentro de uma
pasta ou sub pasta, dever ser digitado o nome da
pasta/ do nome da imagem.

Também podemos carregar imagens externas, basta
apenas copia o link da imagem.

Caso queira, podemos também atribuir classes para
as imagens e fazer alteração por imagens de classes
diferentes.
