Principais formatações

Negrito / Destaque:

nessa frase temos um termo em negrito usando a tag <b> </b> (não semântica).

Nessa frase temos um termo em destaque usando <strong> e </strong>.

Itálico / Ênfase:

E nesta frase temos um termo em italico usando a tag <i> </i>.

Já nasta frase temos um termo em ênfase usando a tag <em> </em> (Semântica).

Texto marcado:

Podemos criar também um texto marcado usando a tag <mark> </mark>.

Texto Grande e pequeno:

Estamos criando um texto grando e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, usando a tag <del> </del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins> </ins>.

existe também o Riscado e o sublinhado com as tags <s> </s> e <u> </u> (não semâticas).
